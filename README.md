# Movie Search App

A React app to search for movies using the OMDB API.

## Features
- Search any movie by name
- Displays movie posters, titles, and release year
- Search on Enter key or button click
- Loading and error states
- Fallback for missing posters

## Tech Used
- React
- Tailwind CSS
- OMDB API
- Vite

## What I learned
- Working with API keys and environment variables
- Handling arrays of data with .map()
- Conditional rendering with &&
- Filtering duplicate results
